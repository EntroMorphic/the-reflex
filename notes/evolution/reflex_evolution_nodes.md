# Nodes of Interest: The Reflex Evolution

## Node 1: The Frozen/Fluid Tension
The Reflexor is frozen (50 nodes, fixed weights). The echip is fluid (routes grow/dissolve, shapes crystallize). This asymmetry feels intentional but unexplained.
**Why it matters:** If everything else follows entropy dynamics, why doesn't the observer?

## Node 2: Silence and Deviation are Dual
The entropy field detects silence (absence of expected signal). The Reflexor detects deviation (presence of unexpected dynamics). These are complements: silence = negative anomaly, deviation = positive anomaly.
**Tension with Node 1:** If they're dual, why are they implemented so differently?

## Node 3: Hebbian Learning as Emergent Attention
AC-CfC uses trained attention for channel weighting. The echip uses Hebbian learning (correlated firing strengthens routes). Hebbian learning IS attention without explicit parameters - it just takes longer to emerge.
**Why it matters:** Attention without training data. Edge-compatible.

## Node 4: Phantom Generation is Underspecified
L3 Imagination produces phantoms (future projections). The topology document doesn't specify how they're seeded. Random? Extrapolation? 
**Connection to entropy field:** High-entropy regions are possibility space. Phantoms should emerge from high entropy, not uniform sampling.

## Node 5: The 3-Level Assumption
The nervous system assumes L1/L2/L3 mapping to cache hierarchy. This couples the architecture to hardware topology.
**Question:** Is this coupling necessary? What breaks if you have 2 levels or 5?

## Node 6: Gradient Flow, Not Just Magnitude
Stigmergy currently broadcasts magnitude (stress level). The entropy field has gradients (direction). Agents could navigate the field, not just sense pressure.
**Why it matters:** Movement enables coordination patterns beyond alarm/calm.

## Node 7: Reflexor as Shape
If the Reflexor became a shape in the echip substrate, it would be subject to crystallization and dissolution. New Reflexors could grow in response to persistent anomaly patterns. Old ones could dissolve when their specialty becomes irrelevant.
**Tension:** Speed. Frozen = fast. Mutable = overhead.

## Node 8: The Spline/CfC Redundancy
Splines: discrete → continuous (interpolation, velocity, prediction). CfC: continuous dynamics with prediction error. Both do prediction. Both handle continuous time.
**Question:** Are they redundant or complementary?

## Node 9: Prediction Error as Entropy Deposit
When the Reflexor fires (prediction error exceeds threshold), that signal could deposit into the entropy field. The field would then track historical anomaly geography, not just current silence.
**Why it matters:** Memory. The field remembers where trouble happened.

## Node 10: Multi-Reflexor Disagreement
Multiple Reflexors with different tau values watching the same signal. Fast Reflexor sees transients. Slow Reflexor sees trends. When they disagree, that's a timescale anomaly.
**Connection to Node 7:** Reflexor specialization through echip dynamics.

## Node 11: The Delta - Boundaries Between Categories
The Laundry Method says check the boundaries. What are the boundaries here?
- Frozen/fluid boundary (Reflexor/echip)
- Silence/deviation boundary (field/chip)
- Discrete/continuous boundary (channels/splines)
- Local/distributed boundary (L1/stigmergy)
**Where mistakes hide:** At these transitions.

## Node 12: Cache Coherency as Entropy Carrier
The 324ns signal already uses cache coherency. The entropy field requires diffusion. Could coherency traffic carry entropy gradients? The hardware is already propagating state.
**Why it matters:** No separate entropy computation needed. Piggyback on existing primitive.
