# Nodes: Applications

## Node 1: Robotics as Throughline
Every vertical is robots in context. Simplifies everything.
**Implication:** One community, one sales motion.

## Node 2: Manipulation
Force feedback, collision response, assembly, tool contact.
**Killer app:** Human-safe collaboration.

## Node 3: Mobile Robots
Obstacle avoidance, cliff detection, bump response, balance.
**Killer app:** Faster navigation without crashes.

## Node 4: Legged Robots
Balance recovery, foot contact, slip response, impact.
**Killer app:** Rough terrain capability.

## Node 5: Drones
Gust response, proximity alert, motor failure, landing.
**Killer app:** Stable flight in wind.

## Node 6: Surgical Robots
Tissue tracking, force limiting, tremor cancellation, heartbeat compensation.
**Killer app:** Precision on moving targets.

## Node 7: Space Robots
Fault recovery, debris avoidance, docking, anomaly detection.
**Killer app:** Autonomy at light-speed delay.

## Node 8: Beachhead Selection
Which application first? Criteria: market size, sales cycle, proof point value.
**Candidates:** Manipulation (biggest), drones (fastest), space (best story).

## Node 9: Cross-Application Patterns
Same Reflex primitives, different configurations.
**Implication:** Build once, deploy everywhere.

## Node 10: Undiscovered Applications
What haven't we thought of? Agriculture? Construction? Entertainment?
**Opportunity:** Let the market tell us.

## Node 11: The Focus Tension
Be everything to everyone vs pick a lane.
**Resolution:** Start narrow, expand with traction.
