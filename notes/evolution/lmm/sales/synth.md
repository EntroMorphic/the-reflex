## Quick Start (Critical)

```bash
# Four commands. Under 5 minutes.
git clone https://github.com/EntroMorphic/the-reflex
cd the-reflex/reflex-robotics
make all
sudo ./build/control_loop
```

**If this doesn't work perfectly, we lose the developer.**
