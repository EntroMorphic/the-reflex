# Nodes: Sales & Deployment

## Node 1: Three Tiers
Open Core (free) → Commercial (subscription) → Premium (custom).
**Why it matters:** Captures full spectrum.

## Node 2: Open Core Value
Ecosystem, prior art, developer love, pipeline.
**Tension:** Giving away value for free.

## Node 3: Commercial Product
Self-organizing Reflexors, Forge SDK, coordination, support.
**This is where money comes from.**

## Node 4: Premium Moat
Certifications + custom forging. High-touch, high-margin.
**Why it matters:** Certification is the moat.

## Node 5: Pricing Philosophy
$2-10K/month commercial. $50-500K premium.
**Principle:** Price reflects value, not cost.

## Node 6: Quick Start Importance
Zero to demo in 30 minutes. Four commands.
**Why it matters:** Friction kills adoption.

## Node 7: Free-to-Paid Conversion
How do open core users become commercial customers?
**Triggers:** Scale, support needs, advanced features.

## Node 8: Upgrade Triggers
Commercial → Premium: certification needs, custom requirements.
**Pattern:** Regulatory pressure drives upgrades.

## Node 9: No Discounting
Price reflects value. Discounting trains bad behavior.
**Tension:** Pressure to close deals.

## Node 10: Competitive Positioning
Compete on capability and trust, not price.
**Why it matters:** Race to bottom = death.

## Node 11: International Pricing
Same price globally? Regional adjustment?
**Question:** What's fair and sustainable?
