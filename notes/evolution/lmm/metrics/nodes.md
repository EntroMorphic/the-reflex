# Nodes: Metrics & ROI

## Node 1: Metrics = Proof
Without metrics, it's faith. With metrics, it's evidence.
**Why it matters:** Justify investment, prove value.

## Node 2: Primary KPIs
Latency P99 < 1μs, Control > 10kHz, Accuracy > 95%, FP < 5%.
**Non-negotiable:** If we miss these, we've failed.

## Node 3: Secondary KPIs
Safety incidents, autonomous resolution, availability, vitality.
**Broader value:** Beyond core function.

## Node 4: Measurement Methods
Instrumented timestamps, cycle counters, validation datasets, manual review.
**Principle:** If you can't measure it, you can't claim it.

## Node 5: Cost Components
Software, hardware, services, internal time.
**Typical total:** $100-200K for production deployment.

## Node 6: Value Components
Prevented collisions, reduced downtime, increased throughput, incidents avoided.
**Challenge:** Counterfactuals are hard.

## Node 7: ROI Calculation
(Value - Cost) / Cost. Usually 50-200%.
**Payback:** Typically 6-12 months.

## Node 8: The Industrial Manipulator Example
$150K cost, $225K value, 50% ROI, 8-month payback.
**Why it matters:** Concrete, credible example.

## Node 9: Baseline Requirement
Measure BEFORE deployment. No baseline = no proof.
**Problem:** Many customers don't track.

## Node 10: The Prevention Paradox
How do you measure what DIDN'T happen?
**Approaches:** Statistical inference, simulation, near-miss tracking.

## Node 11: Customers Who Won't Measure
"We just know it's better." Not proof. Hope.
**Risk:** Can't demonstrate value, can't justify renewal.

## Node 12: ROI Negative Cases
What if the math doesn't work? Walk away?
**Principle:** Honest assessment protects both parties.
