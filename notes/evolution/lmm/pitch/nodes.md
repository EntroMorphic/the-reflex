# Nodes: The Reflex Pitch

## Node 1: The Hot Stove Hook
Universal experience. Immediate connection. "Touch a hot stove, you pull back before you feel pain."
**Why it matters:** Bypasses technical barriers. Everyone understands reflexes.

## Node 2: Brain vs Spine
"Everyone's building brains. We're building spines."
**Tension:** Could sound dismissive of AI. Need to position as complementary.

## Node 3: The 926ns Translation
Raw number is meaningless. "400,000 reactions per blink" is visceral.
**Options:** Blink comparison, nerve impulse comparison, pain-to-reaction comparison.

## Node 4: Before/After Transformation
Before: Robots that hesitate, hurt, need babysitting.
After: Robots that react, protect, survive.
**Why it matters:** Transformation > features.

## Node 5: "Actual Instinct" vs "Artificial Intelligence"
Not simulating. Creating. The Forge proves it.
**Tension:** Bold claim. Need to back it up.

## Node 6: The Competitive Reframe
Everyone else: making robots smarter.
Us: making robots faster.
Smart + slow = dangerous. Fast + smart = safe.
**Why it matters:** Differentiation without competition.

## Node 7: The "Holy Shit" Moment
"The robot stopped before you felt pain."
**Why it matters:** Emotional peak of the pitch.

## Node 8: The Trust Bridge
926ns measured. Skeptical analysis published. Open source core.
**Why it matters:** Credibility for extraordinary claims.

## Node 9: The Call to Action
What do we want them to DO after hearing the pitch?
**Options:** Demo request, pilot discussion, technical deep-dive.

## Node 10: Audience Adaptation
- Executives: Safety, competitive advantage, autonomy
- Investors: Market size, moat, team
- Engineers: Specs, integration, architecture
**Why it matters:** Same core, different emphasis.
