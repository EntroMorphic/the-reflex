# Nodes of Interest: EntroMorphic Lean Business Plan

## Node 1: The Identity Question
Is EntroMorphic a product company, a platform company, or a services company?
- Product: Sell certified Reflexors for specific applications
- Platform: Sell the Forge, let others build Reflexors
- Services: Custom Reflexor development for enterprise clients
**Tension:** Each requires different team, different sales motion, different margins.

## Node 2: The Vertical Focus
Five potential markets: robotics, aerospace, medical, industrial, defense.
Can't serve all at once. Which first?
**Decision criteria:** Speed to revenue, moat creation, reference value, founder expertise.

## Node 3: The Open Core Boundary
Where does "free" end and "paid" begin?
- Too generous: No revenue, competitors benefit
- Too restrictive: No adoption, no ecosystem
**The line:** Primitives open, self-organizing intelligence commercial, certified builds premium.

## Node 4: The Certification Moat
Safety-critical certifications (DO-178C, ISO 26262, IEC 62304) take 18-36 months and significant capital. But once achieved, competitors face the same barrier.
**Question:** Which certification first? Can you start revenue before certification completes?

## Node 5: The Funding Path
Options:
- Bootstrap (slow but retains control)
- Angels/pre-seed (fast but dilutive)
- Strategic investor (aligned but constraining)
- Grants (non-dilutive but slow and bureaucratic)
**Tension:** Deep tech needs patience. Wrong investors demand growth that kills moat-building.

## Node 6: The Team Scaling
Current: Founder + AI collaborators
Needed: Embedded engineers, ML specialists, certification experts, BD
**Question:** Hire before revenue or after? Full-time or contractors?

## Node 7: The Forge as Crown Jewel
The Forge (Delta Observer + crystallization methodology) is the true IP. Not the Reflexor itself - the ability to CREATE Reflexors.
**Strategic implication:** Forge-as-a-Service could be the core business. You don't sell fish; you sell fishing.

## Node 8: The Narrative Capture
"Actual Instinct" is the differentiator. Own the phrase, own the category.
**Action:** TED talk, press, content, thought leadership. Before competitors can claim the same ground.

## Node 9: The Competitive Clock
How long until someone else figures this out?
- Cache coherency insight: 6-12 months
- Full system integration: 24-36 months
**Implication:** Speed matters more than perfection. Ship, certify, establish.

## Node 10: The Pricing Model
Options:
- Per-unit license (traditional)
- Subscription (recurring)
- Usage-based (aligned with value)
- Certification tier (premium for safety-critical)
**Question:** What do customers expect? What maximizes LTV?

## Node 11: The Partnership Strategy
Potential partners:
- Hardware vendors (NVIDIA, Espressif) - distribution
- System integrators (for enterprise sales)
- Research institutions (credibility, talent pipeline)
- Aerospace primes (Lockheed, Northrop) - strategic customers
**Tension:** Partners want exclusivity. Independence wants flexibility.

## Node 12: The Exit Horizon
Is EntroMorphic built to IPO, built to sell, or built to endure?
Different answers lead to different strategies.
**Founder question:** What does success look like in 10 years?
